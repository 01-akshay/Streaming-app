# 🎬 Streaming App

A modern movie streaming web application built using React.js and external movie APIs. Users can browse movies, view details, search for their favorite titles, and enjoy a responsive streaming platform interface.

## 🚀 Features

* Browse trending and popular movies
* Search movies by title
* View movie details and information
* Responsive design for desktop and mobile
* Fast and interactive user interface
* API integration for real-time movie data
* Modern React.js architecture

## 🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Vite
* Movie API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/01-akshay/Streaming-app.git
```

2. Navigate to the project directory

```bash
cd Streaming-app
```

3. Install dependencies

```bash
npm install
```

4. Start the development server

```bash
npm run dev
```
